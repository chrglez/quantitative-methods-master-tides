# Day 1 · Guided exercise

**Topic:** set up your Git/GitHub workflow and take the first steps in R.

## Tasks

1. Fork this repository and clone your fork locally.
2. Open the project in RStudio (double-click the `.Rproj` file).
3. Complete `exercise-template.qmd` — follow the numbered questions.
4. Render the document with `quarto render`.
5. Commit the `.qmd` file (not the rendered outputs) and push to your fork.
6. Confirm on `github.com/<you>/quantitative-methods-master-tides` that
   your commit is visible.

## Deliverable

URL of the committed `.qmd` on your fork.
